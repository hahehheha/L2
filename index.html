Keduanya merupakan bahasa pemrograman yang memiliki pengaruh yang besar pada pengembangan web karena:
  1. CSS di gunakan untuk mengubah penampilan dan tata letak konten suatu web (web jadu estetik dan egonomik/nyaman).
  2. JavaScript berperan menambahkan interaktivitas ke halaman web sehingga perubahan dinamis dapat dialami tanpa memuat ulang halaman

  1. CSS dapat menggunakan dua metode, yaitu metode eksternal (menyambungkan file CSS dan HTML dengan <link rel="stylesheet" href="path file css yang ingin digunakan"> ) dan langsung dalam elemen HTML (menggunakan atribut <style>...</style>).
  2. JavaScript dapat diintegrasikan menggunakan tag <script>...</script>.

  1. Interaktif
  2. Responsive Design
  3. Pengembangan yang efektif dan cepat

  1. Tiap web yang di buat menjadi lebih konsisten
  2. Siap pakai. Selama mengetahui tag/sintaks bootstrap (seperti menggunakan template)
  3. Mudah di pelajari
